# SpaceX Launches

Projeto simples feito com Next.js, TypeScript e TailwindCSS para listar alguns lançamentos da SpaceX usando a API pública.

O foco foi praticar consumo de API, componentes com Next.js e estilização usando Tailwind.  
Cada card mostra o nome, a data e a imagem do lançamento (quando disponível).

## Tecnologias usadas

- Next.js
- TypeScript
- TailwindCSS
- Fetch API
